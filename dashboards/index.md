# 全球趋势仪表盘

本页用于快速查看当前全球趋势状态。保持简洁，只记录需要持续观察的方向。

| 领域 | 当前状态 | 趋势等级 | 简短判断 |
|---|---|---|---|
| AI | 从模型竞争扩展到 Agent、治理、算力和数字主权 | 高 | 模型访问、政策连续性和区域算力正在成为企业选型的重要约束。 |
| 半导体 | AI 算力拉动先进芯片、HBM 和封装需求 | 高 | 供给能力、出口管制和资本开支共同决定产业节奏。 |
| 网络安全 | AI 安全、数据安全和供应链风险同步上升 | 高 | 企业需要把安全前置到模型、数据和自动化流程中。 |
| 宏观经济 | 央行维持高利率，能源冲击使降息路径继续后移 | 高 | 融资成本和通胀不确定性继续约束企业投资与居民消费。 |
| 地缘政治 | 科技竞争、能源通道和供应链安全持续影响市场 | 高 | 地缘政治正在从外部风险变成产业布局变量。 |
| 机器人 | 人形机器人和智能制造进入样机到场景验证阶段 | 中 | 关注成本下降、供应链成熟度和真实应用密度。 |
| 低空经济 | 政策热度高，商业闭环仍在验证 | 中 | 空域、监管、基础设施和运营场景决定落地速度。 |
| 游戏产业 | 硬件入口重新定价，PC 长尾消费和跨媒体 IP 继续升温 | 中高 | 主机平台受成本和利润约束，Steam 折扣生态强化 PC 价格弹性，优质 IP 的跨媒体经营价值上升。 |

## 仪表盘规则

- 每个领域只保留当前状态、趋势等级和一句话判断。
- Dashboard 不写长文，详细分析进入 Reports 或 Topics。
- 当判断发生变化时，同步检查是否需要更新相关 Topic。
