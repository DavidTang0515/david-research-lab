# Game Daily

David Research Lab 的游戏日报栏目，记录经过核实的游戏新闻、国内外玩家态度、初步影响判断和可能的长期趋势信号。

## 栏目边界

- 日报只覆盖游戏及与游戏产业直接相关的内容。
- 默认无剧透。
- 日报属于研究输入层，不代表 Research Lab 的长期结论。
- 普通发售、更新和单款游戏短期口碑只留在日报。
- 平台竞争、商业模式、产业集中度、监管和玩家消费方式等连续变化，可形成趋势信号。
