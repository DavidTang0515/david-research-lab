# Daily Reports

本目录用于存放 David Research Lab 的日常科技趋势简报。

日报是研究输入层，用于记录经过筛选和核实的短期信号。日报不直接替代 Dashboard、Weekly Report 或 Topic，只有经过连续观察、再次核验并具备 6-12 个月保留价值的内容，才进入长期知识沉淀。
