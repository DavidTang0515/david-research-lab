# Daily Reports

本目录为旧版科技日报路径，已不再作为科技趋势日报的发布入口。

自仓库重构后，科技趋势日报统一发布到 `daily/tech/news-YYYY-MM-DD.md`，栏目入口为 `daily/tech/README.md`。本目录仅保留历史文件，不再新增日报。

## 规则

- 新科技日报路径：`daily/tech/news-YYYY-MM-DD.md`。
- 新科技日报栏目首页：`daily/tech/README.md`。
- 周报路径：`reports/YYYY/YYYY-MM-DD-weekXX.md`。
- 日常更新直接 commit 到 `main`，不创建 PR。
