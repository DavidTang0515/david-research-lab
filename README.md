# 科技及经济趋势分析

这是一个长期维护的个人研究仓库，用于沉淀经济、科技、政治、产业和企业数字化转型趋势。

本仓库不定位为新闻收藏夹，而是一个持续演进的研究系统：

> Dashboard 看现在，Reports 记录变化，Topics 沉淀知识。

---

| 入口 | 用途 |

## 三层内容结构

### 1. Dashboard：现在怎么看

回答：当前世界处于什么状态？

用于快速查看：

- AI 热度
- 半导体景气度
- 网络安全重要性
- 宏观风险
- 地缘政治风险
- 对日常生活的影响

### 2. Reports：最近发生了什么

回答：本周、本月、本年发生了什么重要变化？

报告是时间维度的记录，写完后原则上不反复修改。

### 3. Topics：长期怎么看

回答：这个领域到底是什么、怎么演进、长期趋势是什么？

专题是主题维度的知识库，会长期更新。

## 研究原则

1. 少罗列新闻，多解释趋势。
2. 区分事实、判断和假设。
3. 优先引用官方数据、国际组织、公司官方发布和主流媒体。
4. 关注对普通生活、企业经营、数字化转型和产业变化的影响。
5. 不把短期市场情绪误判为长期趋势。
