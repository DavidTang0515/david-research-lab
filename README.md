# 科技及经济趋势分析

本仓库用于沉淀近期经济、科技与政治动态分析，并将宏观趋势、产业变化和投资视角整理为可持续更新的知识库。

## 仓库定位

- 跟踪过去 30-60 天的重要经济、科技、政治事件
- 分析其对日常生活、企业经营和投资方向的影响
- 建立每周/每月可复用的趋势分析模板
- 形成面向个人决策和企业数字化转型观察的资料库

## 当前内容

- `reports/2026-06-10-tech-economy-politics-trends.md`：过去 60 天经济、科技与政治趋势总结
- `reports/2026-06-10-investment-perspective.md`：基于趋势的投资观察与配置建议
- `templates/weekly-trend-report-template.md`：后续周报模板

## 使用原则

1. 优先引用官方数据、国际组织报告、公司官方发布和主流媒体报道。
2. 区分“事实”“判断”“建议”，避免把短期市场情绪误判为长期趋势。
3. 投资内容以趋势分析和资产配置框架为主，不构成具体证券买卖建议。
4. 每期报告建议保留时间范围、核心结论、影响评分和后续观察指标。

## 建议更新频率

- 每周：更新科技产业动态、政策变化、市场情绪和重点行业热度。
- 每月：整理宏观经济数据、地缘政治风险、投资配置建议。
- 重大事件：中美关系、AI监管、央行利率、能源价格、芯片限制等出现明显变化时，可单独更新专题。
